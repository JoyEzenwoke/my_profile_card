# 🚀 Profile Card (Stage 1B)

## 📌 Overview

This project is a **responsive and accessible Profile Card** built using **HTML, CSS, and JavaScript**.

It displays user information and ensures all elements are **testable using `data-testid` attributes**.

---

## ✨ Features

* User name
* Short bio
* Profile image (avatar)
* Current time in milliseconds (`Date.now()`)
* Social media links
* Hobbies list
* Dislikes list

---

## 🧪 Test IDs

All required `data-testid` attributes are included:

* `test-profile-card`
* `test-user-name`
* `test-user-bio`
* `test-user-time`
* `test-user-avatar`
* `test-user-social-links`
* `test-user-hobbies`
* `test-user-dislikes`

---

## ⏱️ Time Functionality

* Displays current time in milliseconds
* Updates every second using JavaScript
* Uses `aria-live="polite"` for accessibility

---

## ♿ Accessibility

* Semantic HTML used (`article`, `figure`, `nav`, `section`)
* Image includes `alt` text
* Links are keyboard accessible
* Focus styles are visible
* External links use `rel="noopener noreferrer"`

---

## 📱 Responsiveness

* Mobile: stacked layout
* Tablet/Desktop: image and content side-by-side
* No layout breaking or overflow

---

## ⚠️ Limitations

* Static data (no backend)
* Uses a fixed image file (`myimage.jpeg`)
* No animations

---

## 🛠️ Technologies

* HTML
* CSS (Flexbox)
* JavaScript
